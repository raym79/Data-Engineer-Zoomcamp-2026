# 1. Data Lakes

## 1.1 What is Data Lake
- Ingest stuctured and unstructured data
- Catalogs and indexes for analysis without data movement
- Stores, secures and protects data at unlimited scale
- Connects data with analytics and machine learning tools

## 1.2 Data Lake v.s. Data Warehouse
| Data Lake | Data Warehouse |
|-----------|----------------|
| Raw       | Rational       |
| Large     | Smaller        |
| Undefined | Refined        |

## 1.3 ETL vs ELT
- Extract Transform and Load vs Extract Load and Transform
- ETL is mainly used for a small amount of data (**data warehouse**) whereas ELT is used for large amounts of data (**data lake**)
- ELT provides data lake support (Schema on read)
